This repository contains code of top 3 winners from Machine Learning Challenge #1. <br />

**Type:** Classification
**Evaluation:** Accuracy
**Duration:**11 days
**Prize:** $500, $300
**Link:** [Machine Learning Challenge 1](https://www.hackerearth.com/challenge/competitive/machine-learning-challenge-one/)
